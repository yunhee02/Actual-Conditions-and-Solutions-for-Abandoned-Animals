# 🐶 유기동물 데이터 분석 프로젝트

## 📖 프로젝트 개요
유기동물 발생 데이터를 분석하여  
발생 현황과 패턴을 파악하고, 문제 해결을 위한 인사이트를 도출한 데이터 분석 프로젝트입니다.

---

## 🎯 분석 목적
- 유기동물 발생 현황 파악
- 지역별 및 시기별 발생 패턴 분석
- 보호 및 관리 정책 개선을 위한 데이터 기반 인사이트 도출

---

## 🛠️ 사용 기술
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

---

## 📊 주요 분석 내용
- 지역별 유기동물 발생 현황 분석
- 시간(연도/월/계절) 기준 발생 추이 분석
- 동물 종류별 발생 비율 분석
- 시각화를 통한 패턴 도출

---

## 💡 결과 및 인사이트
- 특정 시기 및 지역에서 유기동물 발생 집중 현상 확인
- 동물 유형별 발생 차이 확인
- 데이터 기반 관리 및 예방 정책 필요성 도출

---

- 데이터 전처리 및 원본 자료
- 시각화 결과
